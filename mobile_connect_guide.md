# 手机连接校园失物招领系统指南

## 问题说明
错误代码 -7 表示手机无法连接到服务器。USB调试模式不能直接通过USB访问本地服务器，需要使用WiFi局域网。

## 当前网络状态
- 电脑IP: 192.168.43.55
- 网络接口: WLAN (WiFi)
- 子网: 192.168.43.0/24

## 正确的连接方式

### 方法一：WiFi局域网连接（推荐）

**电脑端：**
1. 确保电脑已连接WiFi（当前状态：已连接到 192.168.43.55）
2. 双击运行 `start.bat` 启动服务器
3. 运行 `add_firewall_rule.bat` 确保防火墙允许端口5000

**手机端：**
1. 连接与电脑**同一个WiFi网络**
2. 打开手机浏览器（推荐Chrome或系统自带浏览器）
3. 访问：http://192.168.43.55:5000

### 方法二：USB调试 + 端口转发（高级）

如果您必须使用USB调试，需要配置ADB端口转发：

**步骤：**
1. 手机开启USB调试（前面已说明）
2. 安装ADB工具
3. 运行命令：`adb forward tcp:5000 tcp:5000`
4. 手机浏览器访问：http://localhost:5000

**注意：** 这需要安装Android SDK Platform Tools。

## 故障排查

### 如果仍显示错误 -7：

1. **检查防火墙**
   ```
   运行: check_firewall.bat
   ```

2. **检查服务器是否启动**
   - 确保电脑上Flask服务器正在运行
   - 在电脑浏览器访问 http://127.0.0.1:5000 测试

3. **检查网络连通性**
   ```
   在电脑上 ping 手机IP
   在手机上 ping 192.168.43.55
   ```

4. **临时关闭防火墙测试**
   ```
   控制面板 → Windows Defender 防火墙 → 启用或关闭Windows Defender防火墙
   关闭专用网络防火墙
   ```

## 快速启动

直接运行 `start.bat` 即可启动服务器，然后手机连接同一WiFi访问即可。
